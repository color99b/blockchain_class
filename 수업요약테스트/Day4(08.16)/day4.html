style : 속성. 그 태그의 크기, 외곽선, 위치, 배경 등 다양하게 조정하는 기초적인 방법. 웹페이지를 html안에 다 담으면 보기도 어렵고 크기가 너무 커져버림. CSS 파일, Cascading Style Sheets 로 html파일 외부로 빼버림 
너비나 높이나 숫자를 이용한 수치들에 대한 단위 : px, %, em, rem, vw, vh 정도
px : 한 칸, 한 칸
% : 부모를 기준으로 한 %, 100%는 부모의 너비
em : 글씨 크기를 기준으로 함 (부모 엘레먼트의 글씨 크기임)
rem : 글씨 크기를 기준으로함 (html의 글씨 크기임 -> 모바일,pc 한 번에 만들때 많이 사용)
ex) 글씨크기가 16px 일때 1em, 1rem : 16px
글씨크기가 32px, 1rem - 1em = 32px, 0.5rem = 16px 
            vw, vh : viewport, 브라우저의 크기나 스마트폰의 크기를 기준으로 %
            100vw : 화면의 너비  (viewportwidth)
            
            100vh : 화면의 높이  (viewportheight)
%, vw , vh는 기준이 있고 그 기준의 비율로 너비를 설정하는데,
최소 100px는 가지고 있어야 제대로 나와는 상황이다 -> min-width를 사용
width 설정이랑 똑같음 / 단 max, min은 이미지 조절할 때 정도사용
